Q:Write a program to check whether a number is prime or not?

def prime(n):
    count = 0
    
    for i in range(1,n+1):
        if i%2==0:
            count+=1
    if count==2:
        print(n, "is a prime number")
    else:
        print(n, "is not a prime number")
    

num = int(input("Enter a number:"))
prime(num)
